A simple C++23 module Cmake template with the following directory structure:

```
.
├── CMakeLists.txt
├── lib
│   ├── math.cpp
│   └── math.cppm
├── lib2
│   ├── math2.cpp
│   └── math2.cppm
├── src
│   └── main.cpp
└── src2
    └── main2.cpp

5 directories, 7 files

```

works with gcc only not clang
